# 模块 1：工程基础设施

本模块固定后续网络、协议、存储和持久化模块共同依赖的工程边界。

## 完成内容

- C++20 / CMake 工程与统一告警选项
- 线程安全同步日志，日志过滤路径使用原子级别读取
- `key=value` 配置文件、命令行覆盖和启动前校验
- 无网络依赖的最小测试运行器
- Linux 构建脚本与 GitHub Actions

配置处理始终先加载 `--config`，再应用命令行覆盖，因此参数出现顺序不影响结果。
非法端口、线程数、内存单位和枚举值会在服务启动前失败。

## 验收

```bash
bash scripts/build.sh
bash scripts/run.sh
```
